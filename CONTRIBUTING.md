# Contributing To ClauseFlow

ClauseFlow welcomes focused bug reports, agreement pilots, and improvements to evidence verification.

## Run Locally

```powershell
npm install
npm run dev
```

Use the commands in `README.md` to run contract, frontend, build, and browser checks before opening a pull request.

## Pilot An Agreement

Open a Pilot Request issue with two Bradbury wallet addresses, a public project URL, objective deliverables, acceptance criteria that validators can observe on the web, and a test amount below `0.5 GEN`. Never include a private key, seed phrase, client secret, or `.env` content.

## Good Contributions

- Make transaction status or settlement proof clearer.
- Strengthen source retrieval and criteria-level evidence review.
- Add tests for contract invariants, wallet roles, failures, or indexing delays.
- Improve accessibility and responsive behavior without hiding protocol state.

Unrelated AI helpers, renamed templates, placeholder demos, and cosmetic-only feature bundles are intentionally out of scope.
